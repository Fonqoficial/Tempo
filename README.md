🎼 Tempo | Sistema de Gestión de Eventos MusicalesTempo es una plataforma integral diseñada para músicos y directores de agrupaciones que necesitan centralizar su logística, agenda y repertorio en una única herramienta inteligente.🚀 Características Principales📅 Gestión de Agenda Inteligente (Fase 2 & 3)Timeline Dinámico: Visualización cronológica de próximos eventos con estados de confirmación.Detector de Colisiones: Lógica algorítmica basada en la fórmula $(StartA < EndB) \land (EndA > StartB)$ para evitar solapamientos de horarios por grupo.Vista Dual: Alterna entre un listado detallado (Timeline) y una cuadrícula mensual (Calendario) para planificación a largo plazo.🎵 Biblioteca de Repertorio (Fase 4)Gestión M:N: Relación compleja entre eventos y partituras.Setlists Personalizados: Permite asignar un orden específico de interpretación para cada bolo.Acceso Directo a Partituras: Visualización inmediata de archivos PDF asociados a cada obra.🔍 Optimización de Flujo (Fase 5)Filtros Avanzados: Búsqueda en tiempo real por nombre de evento, lugar o agrupación musical.Diseño Responsive: Interfaz optimizada con Tailwind CSS para uso en movilidad (tablets y smartphones durante ensayos).🛠️ Stack TecnológicoFrontend: React 18 con Vite para una experiencia de desarrollo ultrarrápida.Lenguaje: TypeScript para un código robusto y tipado.Base de Datos & Auth: Supabase (PostgreSQL) con políticas de seguridad RLS.Iconografía: Lucide React.Manejo de Fechas: date-fns.
